# Start Board

VSCodeの空のウィンドウにカスタマイズ可能なダッシュボードを表示する拡張機能です。

## 機能

### プロジェクトクイックアクセス

- よく使うプロジェクト（フォルダ・ワークスペース）をダッシュボードに登録
- ワンクリックでプロジェクトを開く
- 存在しないパスを検出し、リストから削除可能

### ASCIIアートカスタマイズ

- ダッシュボードにASCIIアートを表示
- テキスト、フォント、サイズ、行間をカスタマイズ可能

### 自動表示

- VSCode起動時、フォルダやワークスペースが開かれていない場合に自動でダッシュボードを表示

## 使い方

### プロジェクトの追加

1. プロジェクトを開いた状態でコマンドパレット（`Ctrl+Shift+P` / `Cmd+Shift+P`）を開く
2. `Start Board: 現在のプロジェクトを追加` を実行
3. ダッシュボードにプロジェクトが追加されます

### ダッシュボードの表示

- コマンドパレットから `Start Board: ダッシュボードを表示` を実行

### ASCIIアートのカスタマイズ

設定から以下の項目をカスタマイズできます：

- `startBoard.asciiArt.text`: 表示するテキスト（複数行対応）
- `startBoard.asciiArt.fontFamily`: フォントファミリー
- `startBoard.asciiArt.fontSize`: フォントサイズ（px）
- `startBoard.asciiArt.lineHeight`: 行の高さ

## 要件

- Visual Studio Code 1.85.0 以上
