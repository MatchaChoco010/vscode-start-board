# Product Overview

VSCodeの空のウィンドウにカスタマイズ可能なダッシュボードを表示する拡張機能。プロジェクトへの素早いアクセスとASCIIアートによるパーソナライズを提供します。

## Core Capabilities

- **空ウィンドウ自動検出**: VSCode起動時、フォルダやワークスペースが開かれていない場合に自動でダッシュボードを表示
- **プロジェクト管理**: プロジェクト（フォルダ/ワークスペース）の追加、一覧表示、削除、クイックオープン
- **ASCIIアート表示**: カスタマイズ可能なテキストとフォント設定でダッシュボードをパーソナライズ
- **パス検証**: 存在しないパスを検出し、リストからの削除オプションを提供

## Target Use Cases

- 複数のプロジェクト間を頻繁に切り替える開発者
- VSCode起動時に素早く目的のプロジェクトを開きたいユーザー
- ワークスペースのスタートページをカスタマイズしたいユーザー

## Value Proposition

VSCodeの標準的な「最近開いたフォルダ」機能を補完し、より視覚的で管理しやすいプロジェクトランチャーを提供します。ASCIIアートによる視覚的なカスタマイズ性と、フォルダ/ワークスペース両対応による柔軟性が特徴です。
